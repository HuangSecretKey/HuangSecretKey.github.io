abaqus2018安装教程

发表于 2021-08-14 分类于 abaqus   |   字数统计: 1.7k(字)   |   阅读时长≈ 5(分)   |   阅读次数： Valine：

本文为转载文章，具体链接已放在最后

下面开始安装。

安装准备

1.下载安装包Abaqus2018.zip。

2.解压缩至文件夹Abaqus2018，该文件夹下含有一个文件夹abaqus 2018和一个文件Read Me.txt。

解压后的文件夹

abaqus 2018文件夹下有我们需要的安装程序和破解程序。

安装和破解程序文件夹

3.新建安装文件夹Abaqus以安装程序。

为了很好的区分解压缩文件夹和安装文件夹，我这里用不同的名字表示，Abaqus2018是解压缩文件夹，Abaqus是安装文件夹。Abaqus2018文件夹是从zip压缩包里面解压出来的，Abaqus是即将要安装程序的位置，现在是空的。

安装本地服务器

1.找到Abaqus2018\abaqus 2018\破解文件_SolidSQUAD_\SSQ\SolidSQUAD_License_Servers文件夹，将其复制到Abaqus文件夹下。

复制SolidSQUAD_License_Servers文件夹

2.找到Abaqus2018\abaqus 2018\破解文件_SolidSQUAD_\Vendors文件夹并将其复制到Abaqus\SolidSQUAD_License_Servers文件夹下

复制Vendors文件夹

期间会提示确认文件夹替换，点击“是”即可。

确认文件夹替换

3.在Abaqus\SolidSQUAD_License_Servers文件夹下找到install_or_update.bat文件，右击以管理员权限运行。

以管理员权限运行install_or_update.bat文件

看到以下界面，就ok了。

本地服务器安装成功啦

设置环境变量

新建系统变量NOLICENSECHECK，并设置其值为true。

设置环境变量

安装程序

1.以管理员权限运行Abaqus2018\abaqus 2018\DS.SIMULIA.SUITE.2018.WIN64\1\setup.exe

管理员权限运行setup.exe

由于只涉及到Abaqus，所以其他的就先不装了，后期需要安装的，自行去Abaqus2018\abaqus 2018\DS.SIMULIA.SUITE.2018.WIN64\2文件夹下安装。

看好下图中安装的，按住Ctrl并左击选择或取消选择。

只装这4个，看好是哪4个

选好之后点击“下一步”。

2.在弹出的对话框中安装Product Documentation，因为安装的是Product Documentation，所以我这里安装在Abaqus\Documentation目录下，Documentation文件夹自己新建。然后下一步。

安装Product Documentation

考虑到后期会加装插件，所以这里把文档全部安装，直接下一步。

安装的文档

确认安装信息，然后安装。（我这边可能多一个安装标识，因为之前安装过一次，为了写这个教程，卸载重装了，所以会有安装标识以区分）

确认安装信息

等待安装成功即可。

等待安装成功

安装成功点击关闭。

安装后关闭

3.在弹出的对话框中安装SimulationServices，因为安装的是SimulationServices，所以我这里安装在Abaqus\SimulationServices\V6R2018x目录下，SimulationServices和V6R2018x目录可自己新建，也可直接修改给定的默认目录为自己的安装文件夹。然后下一步。

选择安装的目录

选择安装的部件，这里全选，直接下一步。

即将安装的部件

确认安装后，直接安装。

确认安装，点击安装

等待安装成功。

等待安装成功

安装成功后关闭。

关闭即可

4.在弹出的对话框中安装CAA API，由于前面已经装好了SimulationServices，系统会自动找到SimulationServices，直接下一步即可。

直接下一步

选择需要安装的部件，这里全选，直接下一步。

全选需要安装的部件

确认部件以安装，点击安装即可。

确认部件以安装

等待安装成功。

等待安装成功

安装成功后关闭。

关闭就行了

5.在弹出的对话框中安装CAE，我这边选择安装到Abaqus\SIMULIA\CAE\2018，SIMULIA、CAE和2018这几个目录可以自己新建，也可直接修改给定的默认目录为自己的安装文件夹。然后下一步。

安装CAE的目录

在接下来的对话框中选择第一项，然后下一步。

选择FLEXnet

在License Server 1的文本框中输入：27800@localhost，这里的localhost为你的计算机名称，计算机名称去计算机的属性里面找。我这边是27800@PC-20180927BAPT，然后下一步。

输入License Server 1

自动识别SimulationServices的路径，直接下一步即可。

自动识别SimulationServices ，下一步

为了目录的统一，我把Commands安装到之前设置好的\SIMULIA文件夹下。如下图所示。然后下一步。

Commands安装目录

接下来是plugins的安装，为了目录统一，还是安装到之前的SIMULIA\CAE\plugins\2018目录下。如下图所示。然后下一步。

plugins的安装目录

接下来选择第一个就可以。然后下一步。

选择第一个，然后下一步

为了目录统一，安装到Abaqus\temp下，如下图所示。然后下一步。

安装工作目录

确认安装信息以安装。

确认安装信息并安装

等待安装成功。

等待安装成功

还是等待安装成功

期间会弹出防火墙的提醒，放行就行。然后就会有验证结果。Continue

验证结果

安装成功后关闭就行。

成功后关闭

至此，已经全部安装完成啦。关闭即可。

全部安装完成

验证安装

找到安装的Abaqus

找到程序

然后打开，会有CMD窗口，不用管它就行。

不用管这个窗口

然后就是我们需要用到的软件啦。

Abaqus启动界面

尽情享用吧！！！

（PS：可以把解压缩的文件夹Abaqus2018删除）

2019.10.28更新

小明同学在安装后打不开，报以下错误。

打开软件报错

看错误是atio6axx.dll动态链接库引发的，去查找了一下atio6axx.dll，简介如下：

atio6axx.dll简介

原来是显卡的问题，这台电脑是A卡，那我把A卡禁了，直接用集显就行了。
